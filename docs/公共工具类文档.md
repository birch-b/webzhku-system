## 三、详细说明

### 3.1 DBUtil - 数据库连接池工具

**文件路径**: `src/main/java/com/taobao/util/DBUtil.java`

**功能**: 使用 Druid 连接池管理数据库连接，提供统一的连接获取和资源关闭方法。

**核心方法**:

| 方法名 | 返回值 | 参数 | 说明 |
|--------|--------|------|------|
| getConnection() | Connection | 无 | 获取数据库连接 |
| close(Connection, PreparedStatement, ResultSet) | void | conn, ps, rs | 关闭连接、语句、结果集 |
| close(Connection, PreparedStatement) | void | conn, ps | 关闭连接和语句 |
| getDataSource() | DataSource | 无 | 获取数据源 |

**使用示例**:
```java
Connection conn = null;
PreparedStatement ps = null;
ResultSet rs = null;
try {
    conn = DBUtil.getConnection();
    ps = conn.prepareStatement("SELECT * FROM user WHERE id = ?");
    ps.setLong(1, userId);
    rs = ps.executeQuery();
    // 处理结果集...
} catch (SQLException e) {
    e.printStackTrace();
} finally {
    DBUtil.close(conn, ps, rs);
}
```

**配置方式**:
- 默认配置（无 db.properties 时）：连接 `localhost:3306/taobao_shop`，用户名 `root`，密码 `root`
- 自定义配置：在 `src/main/resources/db.properties` 中配置

### 3.2 MD5Util - MD5加密工具

**文件路径**: `src/main/java/com/taobao/util/MD5Util.java`

**功能**: 提供字符串的 MD5 加密功能，用于密码存储。

**核心方法**:

| 方法名 | 返回值 | 参数 | 说明 |
|--------|--------|------|------|
| md5(String) | String | str: 待加密字符串 | 返回32位MD5加密结果 |

**使用示例**:
```java
String password = "123456";
String encrypted = MD5Util.md5(password);
// encrypted = "e10adc3949ba59abbe56e057f20f883e"
```

### 3.3 StringUtil - 字符串处理工具

**文件路径**: `src/main/java/com/taobao/util/StringUtil.java`

**功能**: 提供常用字符串处理方法，如空值判断、字符串截取等。

**核心方法**:

| 方法名 | 返回值 | 参数 | 说明 |
|--------|--------|------|------|
| isEmpty(String) | boolean | str: 字符串 | 判断字符串是否为空 |
| isNotEmpty(String) | boolean | str: 字符串 | 判断字符串是否非空 |
| trim(String) | String | str: 字符串 | 去除首尾空格，空值返回空串 |
| substring(String, int) | String | str, len: 字符串和长度 | 安全截取字符串，超出长度返回原串 |

**使用示例**:
```java
if (StringUtil.isEmpty(username)) {
    // 用户名不能为空
}
String name = StringUtil.trim(inputName);
String shortName = StringUtil.substring(fullName, 20);
```

### 3.4 DateUtil - 日期时间工具

**文件路径**: `src/main/java/com/taobao/util/DateUtil.java`

**功能**: 提供日期格式化、日期计算等常用日期处理方法。

**核心方法**:

| 方法名 | 返回值 | 参数 | 说明 |
|--------|--------|------|------|
| format(Date) | String | date: Date对象 | 格式化为 "yyyy-MM-dd HH:mm:ss" |
| format(Date, String) | String | date, pattern: Date对象和格式 | 自定义格式化 |
| parse(String) | Date | str: 日期字符串 | 解析 "yyyy-MM-dd HH:mm:ss" |
| parse(String, String) | Date | str, pattern: 字符串和格式 | 自定义解析 |
| now() | Date | 无 | 获取当前时间 |
| getToday() | Date | 无 | 获取今天日期（时间部分为0） |

**使用示例**:
```java

// nowStr = "2026-07-12 15:30:45"

Date date = DateUtil.parse("2026-01-01 00:00:00");
```

### 3.6 UploadUtil - 文件上传工具

**文件路径**: `src/main/java/com/taobao/util/UploadUtil.java`

**功能**: 通用文件上传工具，支持文件保存、目录创建等。

**核心方法**:

| 方法名 | 返回值 | 参数 | 说明 |
|--------|--------|------|------|
| saveFile(InputStream, String) | String | inputStream, filePath | 保存文件流到指定路径 |
| generateFileName(String) | String | originalName: 原文件名 | 生成唯一文件名（时间戳+随机数） |
| ensureDirectoryExists(String) | void | dirPath: 目录路径 | 确保目录存在，不存在则创建 |

**使用示例**:
```java

String fileName = UploadUtil.generateFileName("phone.jpg");
// fileName = "20260712153045_12345.jpg"

UploadUtil.saveFile(inputStream, uploadDir + fileName);
```

### 3.7 FileUploadUtil - Servlet文件上传工具

**文件路径**: `src/main/java/com/taobao/util/FileUploadUtil.java`

**功能**: 基于 Apache Commons FileUpload 的 Servlet 文件上传专用工具。

**核心方法**:

| 方法名 | 返回值 | 参数 | 说明 |
|--------|--------|------|------|
| parseRequest(HttpServletRequest) | Map<String, Object> | request: HTTP请求 | 解析上传请求，返回表单字段和文件 |
| saveUploadedFile(FileItem, String) | String | fileItem, uploadDir | 保存上传的文件项 |

**使用示例**:
```java
Map<String, Object> result = FileUploadUtil.parseRequest(request);
String productName = (String) result.get("productName");

List<FileItem> files = (List<FileItem>) result.get("files");
for (FileItem file : files) {
    String filePath = FileUploadUtil.saveUploadedFile(file, "/upload/product/");
}
```

### 3.8 JsonUtil - JSON序列化工具

**文件路径**: `src/main/java/com/taobao/util/JsonUtil.java`

**功能**: 提供对象与JSON字符串之间的转换功能。

**核心方法**:

| 方法名 | 返回值 | 参数 | 说明 |
|--------|--------|------|------|
| toJson(Object) | String | obj: Java对象 | 对象转JSON字符串 |
| fromJson(String, Class) | T | json, clazz: JSON字符串和目标类型 | JSON字符串转Java对象 |
| toJsonArray(List) | String | list: 列表对象 | 列表转JSON数组 |

**使用示例**:
```java

List<String> parsed = JsonUtil.fromJson(json, List.class);
```

## 四、过滤器说明

### 4.1 EncodingFilter - 编码过滤器

**文件路径**: `src/main/java/com/taobao/filter/EncodingFilter.java`

**功能**: 设置全局请求和响应的字符编码为 UTF-8，防止中文乱码。

**配置**: 在 `web.xml` 中配置，拦截所有请求（`/*`）。

### 4.2 AuthFilter - 统一权限过滤器

**文件路径**: `src/main/java/com/taobao/filter/AuthFilter.java`

**功能**: 统一处理用户登录认证和角色权限检查。

**权限映射**:

| 路径前缀 | 允许角色 | 说明 |
|----------|----------|------|
| /admin/* | operator | 运营商后台 |
| /shop/* | shopkeeper | 商家后台 |
| /customer/*, /cart/*, /order/*, /payment/*, /review/*, /aftersale/*, /address/* | customer | 顾客前台 |
| /product/*, /announcement/*, /shop/home, /shop/product/* 等 | browser/customer/shopkeeper/operator | 公开页面 |

**拦截逻辑**:
1. 放行静态资源
2. 放行登录/注册/登出入口
3. 放行公开页面（浏览者可访问）
4. 检查是否登录，未登录跳转登录页
5. 检查角色权限，越权访问跳转到对应登录页

### 4.3 ShopAuthFilter - 商家权限过滤器

**文件路径**: `src/main/java/com/taobao/filter/ShopAuthFilter.java`

**功能**: 在 AuthFilter 之后，进一步检查商家店铺状态。

**检查逻辑**:
- shopStatus = -2: 未开店，跳转开店申请页
- shopStatus = 0: 审核中，显示审核中页面
- shopStatus = -1: 已关闭，显示关闭页面
- shopStatus = 1: 营业中，放行

## 六、工具类使用规范

### 6.1 调用方式
所有工具类均为静态方法，直接通过类名调用，无需实例化。

### 6.2 异常处理
- `DBUtil` 获取连接失败会抛出 RuntimeException，调用方需捕获处理
- 文件上传相关方法可能抛出 IOException，需捕获处理

### 6.3 代码规范
- 工具类方法应为静态方法（static）
- 方法命名应清晰，遵循驼峰命名法
- 参数和返回值应使用泛型增强类型安全

### 6.4 新增工具类流程
1. 在 `com.taobao.util` 包下创建新工具类
2. 编写工具方法，确保方法为静态
3. 在本文档中添加说明
4. 提交到公共基础分支
